# School Security Management Web Application
## Overview
This is a web-based application designed to streamline security operations within a school. It provides tools for managing incidents, tracking visitor access, and maintaining staff records, all accessible through a user-friendly interface.
## Features
*   **Incident Reporting:** Submit and view details of security incidents via a web form.
*   **Visitor Log:** Record and retrieve visitor information (name, purpose, time in/out).
*   **Security Personnel Database:** Manage a directory of security staff with their contact and shift details.
*   **Interactive Interface:** User-friendly forms and displays built with HTML, CSS, and JavaScript.
## Technologies Used
*   **Backend:** PHP (for server-side logic and database interaction)
*   **Frontend:**
    *   HTML (for structuring web pages)
    *   CSS (for styling and layout)
    *   JavaScript (for interactive elements and dynamic content)
*   **Database:** MySQL (or similar relational database, typically used with PHP)
## Installation (Local Setup)
1.  **Web Server & PHP:** Ensure you have a web server installed (like Apache or Nginx) with PHP (version 7.4 or higher recommended). XAMPP or MAMP are great all-in-one solutions for local development.
2.  **Database:** Set up a MySQL database. Create a new database (e.g., `school_security_db`).
3.  **Import Database Schema:** Import the provided SQL file (e.g., `database.sql`) into your new MySQL database. This will create the necessary tables.
